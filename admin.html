⚙️ Outlook Sync — Azure instellen

Voer je Microsoft Azure Client ID in. Eenmalig instellen, daarna werkt de sync automatisch.

📋 Eenmalige setup (5 minuten):
1. Ga naar portal.azure.com → log in met je Microsoft account
2. Zoek "App registrations" → klik + New registration
3. Naam: Diasco Portaal | Account type: Personal Microsoft accounts
4. Redirect URI: https://diasco3.vercel.app/admin.html (type: SPA)
5. Klik Register → kopieer de Application (client) ID
6. Ga naar API permissions → Add a permission → Microsoft Graph → Delegated → Calendars.ReadWrite
7. Plak de Client ID hierboven en klik Opslaan

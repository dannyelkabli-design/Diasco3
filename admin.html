Database instellen

Verbind Supabase voor onbeperkte en gedeelde opslag. Eenmalige setup van ~5 minuten.

Stap 1 — Supabase project aanmaken

  1. Ga naar supabase.com en maak een gratis account aan
  2. Klik New Project → geef het een naam (bijv. "diasco")
  3. Wacht tot het project klaar is (~1 min)
  4. Ga naar Project Settings → API
  5. Kopieer de Project URL en anon public key

Stap 2 — Tabellen aanmaken

Kopieer en voer deze SQL uit in de Supabase SQL Editor:

-- Afspraken tabel CREATE TABLE IF NOT EXISTS afspraken ( id uuid PRIMARY KEY DEFAULT gen_random_uuid(), fname text, lname text, email text, phone text, date text NOT NULL, time text NOT NULL, treatment text, notes text, created_at timestamptz DEFAULT now() ); -- Geblokkeerde slots tabel CREATE TABLE IF NOT EXISTS blocked_slots ( id uuid PRIMARY KEY DEFAULT gen_random_uuid(), date text NOT NULL, time text NOT NULL, reason text DEFAULT 'Geblokkeerd', created_at timestamptz DEFAULT now(), UNIQUE(date, time) ); -- Row Level Security uitschakelen (intern gebruik) ALTER TABLE afspraken ENABLE ROW LEVEL SECURITY; ALTER TABLE blocked_slots ENABLE ROW LEVEL SECURITY; CREATE POLICY "allow_all" ON afspraken FOR ALL USING (true) WITH CHECK (true); CREATE POLICY "allow_all" ON blocked_slots FOR ALL USING (true) WITH CHECK (true);

Stap 3 — Verbinding testen

Klik op "Verbinding testen" om te controleren of alles werkt...
